## 🎯 工作路线图

当前系统架构：

```
摄像头 → SegFormer 分割 → 航道检测 → Pure Pursuit → PID 控制 → 控制信号输出
```

系统输出的控制信号字段：
- `steering_deg`: 舵角 (-30 ~ +30 度)
- `throttle_pct`: 油门百分比 (0 ~ 100%)
- `speed_mps`: 目标速度 (m/s)
- `heading_deg`: 航向角
- `target_x_m`: 前向目标距离
- `target_y_m`: 横向目标偏移
- `total_mileage_m`: 总路径里程

**预计工期**：数据标注(5天) + 模型训练(4天) + 测试优化(3天) = 约2周
