# 快速开始指南

## 🚀 立即开始数据标注

### 第一步：安装Labelme

```bash
# Windows
pip install labelme

# 验证安装
labelme --version
```

### 第二步：准备数据

```bash
# 检查你的图像数据
cd c:\Users\15181\Desktop\boat
dir dataset\

# 应该看到大量 .jpg 文件
```

### 第三步：开始标注

```bash
# 启动Labelme
labelme dataset\ --output dataset\annotations --labels labels.txt

# 或者简化版
labelme dataset\
```

**标注快捷键**：
- `Ctrl+鼠标左键`：创建多边形点
- `双击`：完成当前多边形
- `D`：下一张图片
- `A`：上一张图片  
- `Ctrl+S`：保存
- `Ctrl+E`：编辑模式
- `Ctrl+J`：编辑标签

### 第四步：标注规范

为每张图像标注**3个类别**：

1. **background**（背景）- 黑色
   - 天空、树木、建筑物、岸边植被等
   
2. **water**（水域）- 蓝色  
   - 可以航行的水面区域
   - 包括倒影
   
3. **boundary**（边界）- 红色
   - 河岸线、护栏、堤坝
   - 其他船只、障碍物

**标注顺序建议**：
1. 先标注大的背景区域
2. 再标注水域
3. 最后标注边界（会覆盖其他区域）

### 第五步：质量检查

标注完一部分后（建议每20-30张检查一次）：

```bash
python scripts\check_annotations.py --dir dataset\annotations
```

假设每张图像标注需要**3-5分钟**：

- 每标注1小时休息10分钟
- 保持标注一致性比速度更重要
- 遇到困难的图像可以标记跳过

## ⚡ 标注完成后

### 1. 转换标注格式

```bash
python scripts/labelme_to_mask.py --input dataset/annotations --output dataset/masks --images dataset/images
```

### 2. 划分数据集

```bash
python scripts/split_dataset.py --images dataset/images --masks dataset/masks --output dataset_final
```

### 3. 验证数据集

```bash
dir dataset_final\images\train
dir dataset_final\masks\train
```

应该看到：
- `dataset_final\images\train\` - 训练图像
- `dataset_final\images\val\` - 验证图像  
- `dataset_final\images\test\` - 测试图像
- `dataset_final\masks\train\` - 训练掩码
- `dataset_final\masks\val\` - 验证掩码
- `dataset_final\masks\test\` - 测试掩码

## 🎯 标注示例

### 好的标注 ✅
- 边界线清晰准确
- 覆盖整个图像
- 类别区分明确
- 细节处理到位

### 不好的标注 ❌
- 边界模糊不清
- 有大片未标注区域  
- 类别混淆
- 粗糙马虎

## 💡 标注技巧

1. **放大图像**：在边界处放大图像以获得更精确的标注
2. **使用多边形**：不要用太多点，关键转折点即可
3. **保持一致**：相似的场景使用相同的标注策略
4. **处理困难样本**：
   - 光照不均：按实际类别标注
   - 倒影：标注为水域
   - 模糊区域：按最可能的类别
