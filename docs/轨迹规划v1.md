## 世界坐标系与里程记录

### 1. 坐标系定义

采用**船体坐标系**，原点位于相机正下方水面点：

```
          图像坐标系                    世界坐标系（米）

         (0, 0)                      Y（前进方向）
           ↑                            ↑
           │ v                          │ +y_m
           │                            │
    ◄──────┼──────►  u           ◄──────┼──────►  X
     (0,cy)  (cx,cy) (w,cy)      -x_m  (0,0)  +x_m
           │                            │
    (cx, h-1) = 船的位置         相机正下方水面 = 原点
```

- **x_m**：横向偏移（+右 -左），单位：米
- **y_m**：前方距离 / 累计里程，单位：米
- **Z**：向上（水面 = 0，相机 = +h）

### 2. IMX390-GMSL2 相机参数

### 3. 像素→世界坐标换算

#### 3.1 精确模式：IMX390 针孔相机投影（推荐）

基于相机内参 + 安装外参的透视逆投影，精确处理近大远小的非线性缩放。

```
                    相机
                   ╱ │ ╲
                 ╱   │h  ╲        h = camera_height
               ╱     │     ╲      θ = camera_pitch
             ╱ θ     │       ╲
    ────────╱────────┼────────╲──── 水面 (Z=0)
          近处     正下方     远处
        (像素大)            (像素小)
```

**数学推导**（针孔模型）：

```
已知: fx = 1920 / (2·tan(HFOV/2))    # 焦距（像素单位）
      fy = fx                          # 方形像素

对于像素 (u, v):
  dx = (u - cx) / fx                   # 归一化相机坐标
  dy = (v - cy) / fy

  世界方向向量:
    d_X = dx
    d_Y = dy·sin(θ) + cos(θ)           # 前方分量
    d_Z = dy·cos(θ) - sin(θ)           # 垂直分量

  与水面 (Z=0) 交点:
    t = h / (sin(θ) - dy·cos(θ))       # 当 d_Z < 0 时有解

  世界坐标:
    x_m = t · dx                        # 横向偏移
    y_m = t · (dy·sin(θ) + cos(θ))      # 前方距离
```

**优势**：
- 正确处理透视缩放：图像上方 1px = 远处多米，下方 1px = 近处少米
- 不依赖 mask 中的河道宽度测量
- 只需 3 个安装参数：高度、俯仰角、镜头HFOV

#### 3.2 后备模式：河宽比例法

当不提供相机安装参数时自动启用，利用已知河宽 (2.5m) 和每行河道像素宽度做线性近似：

```
scale = 2.5m / river_width_px(row)
x_m = (x_px - img_cx) × scale
y_m = Σ(逐段像素距离 × 局部scale)
```

### 7. 安装参数测量指南

```
                ┌─────────┐
                │ IMX390  │ ← 相机
                │ camera  │
                └────┬────┘
                     │
                     │ camera_height (垂直距离)
                     │
      ═══════════════╧═══════════════ 水面
                  ↗
          camera_pitch (与水平线的夹角)
```

- **camera_height**：用卷尺测量相机镜头中心到静水面的垂直距离（米）
- **camera_pitch**：用手机水平仪 App 贴在相机上测量，或根据视野范围估算
- **camera_hfov**：查看镜头型号确定（H60=63.9°, H120=120.6°）

### 8. 两种模式对比

| 对比项 | 相机投影模式 | 河宽比例法 |
|--------|------------|-----------|
| 精度 | 高（物理模型） | 中（线性近似） |
| 透视处理 | 精确非线性 | 逐行线性 |
| 依赖参数 | 高度+俯仰+FOV | 河宽 |
| 适用场景 | 已知相机安装 | 快速验证 |
| 远距离精度 | 好 | 偏差较大 |
