# 代码精简 v1 - 移除 MAVLink/PWM/ROS 相关代码

## 📌 背景

项目之前包含了 MAVLink 飞控通信、PWM 信号转换和 ROS2 节点等代码。根据需求，**飞控对接和控制信号输出由同事负责**，本项目只需负责感知→规划→输出控制信号三个环节。因此进行了全面的代码精简。

**精简目标**：
- 移除所有 MAVLink 和 PWM 相关代码
- 移除所有 ROS2 相关代码和节点
- 保留感知（SegFormer）、规划（Pure Pursuit）、控制信号输出（PID）
- 保持代码模块化和可独立测试
